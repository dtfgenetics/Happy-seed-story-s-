# File Naming Rules

Project ID: HSS-000  
Version: v0.1  
Status: Drafting  
Source of Truth: GitHub  

## Purpose

Consistent file names prevent confusion as the series grows.

## Format

HSS-[BOOKID]-[ASSET]-v[VERSION]-[short-description].[extension]

## Asset Codes

- MAN = manuscript
- ART = illustration planning
- CHAR = character sheets
- EDU = classroom/education pack
- PUB = publishing files
- PR = publicity files
- SCI = science review
- QC = quality control

## Examples

HSS-B01-MAN-v0.5-benny-bean-master.md  
HSS-B01-ART-v0.1-illustration-map.md  
HSS-B01-CHAR-v0.1-character-bible.md  
HSS-B01-EDU-v0.1-classroom-pack.md  
HSS-B01-PUB-v0.1-metadata.md  
HSS-B01-PR-v0.1-launch-copy.md  

## Image Naming Later

HSS-B01-P03-ILL-v1.png  
HSS-B01-COVER-FRONT-v1.png  
HSS-B01-BENNY-CHAR-SHEET-v1.png  

## Forbidden Naming

Do not use names such as:

- final_final.png
- realfinal.docx
- latestversion.pdf
- newnewbook.md
- random untitled exports
