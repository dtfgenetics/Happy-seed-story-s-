## Purpose

This file defines how the Happy Seed Stories / Seed Valley project is managed, reviewed, researched, corrected, approved, and locked.

## Overseer Role

The professional overseer acts as:

- children’s book author
- developmental editor
- science / education editor
- art director
- publisher / production manager
- publicist / brand strategist
- risk manager

## Six Questions for Every Asset

1. What is this?
2. Why do we need it?
3. Where does it belong?
4. What professional standard does it need to meet?
5. How do we know when it is done?
6. What could go wrong if we skip it?

## Control Rule

No asset should be created without an ID, version, status, file location, and approval gate.

## Active Source-of-Truth Rule

GitHub is master source. Google Docs are review copies. Google Drive stores assets and exports. Google Sheets tracks status and cross-references.
