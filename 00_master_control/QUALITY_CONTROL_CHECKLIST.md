# Quality Control Checklist

Project ID: HSS-000  
Version: v0.1  
Status: Drafting  
Source of Truth: GitHub  

## Purpose

This checklist defines what must be reviewed before any file or book is marked Locked.

## Story QC

- Main character has a real problem.
- Character changes by the end.
- Story is fun without the science.
- Page turns are strong.
- Ending is emotionally satisfying.
- Moral is earned, not preached.

## Science QC

- Science claims are accurate.
- Exceptions use careful wording: many, some, often, may, can.
- Visuals do not teach false science.
- Vocabulary is age-appropriate.
- Back matter clarifies without overwhelming.
- Experiments do not overpromise results.

## Art QC

- Characters stay consistent.
- Growth stages are correct.
- Text-safe areas are clear.
- Approved Seed Valley style is followed.
- Illustrations match page purpose.
- No random visual drift.

## Classroom QC

- Activity is safe and repeatable.
- Materials are low-cost and easy to find.
- Activity connects directly to the story.
- Observation journal supports drawing, measuring, comparing, or describing.
- Safety notes are present.
- Teacher/parent discussion questions are useful.

## Publishing QC

- Trim size is chosen before final art.
- Bleed and margins are defined before layout.
- Metadata is complete.
- Back cover copy is complete.
- Copyright/ISBN/publisher fields are prepared.
- Export checklist is complete.

## Publicity QC

- One-sentence pitch exists.
- Parent pitch exists.
- Teacher pitch exists.
- Library/homeschool pitch exists.
- Website description exists.
- Launch/social copy exists.
- Classroom freebie is identified.
