## Purpose

This file translates the Book 1 working manuscript into a visual production guide. It is not final image generation yet. It is the blueprint for later illustration prompts.

## Global Style Anchor

Warm premium early-reader storybook style, colorful Seed Valley world, rounded friendly characters, expressive faces, soft natural lighting, cozy magical plant-science environment, clear emotional storytelling, accurate germination sequence.

## Global Visual Continuity

Benny growth sequence:
closed seed -> damp/swollen seed -> cracked seed -> first root downward -> shoot upward -> tiny sprout.

Do not reverse this sequence.
