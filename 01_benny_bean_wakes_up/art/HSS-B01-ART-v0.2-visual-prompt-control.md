## Purpose

This file controls how future image prompts are created for Book 1. It prevents random visuals, science drift, character drift, and layout problems.

## Prompt Status

Final image prompts are not approved yet.

Allowed now:
- concept prompt planning
- style exploration prompts
- character prompt blocks
- page composition planning
- text-safe area planning

Not allowed yet:
- final page image generation
- final cover image generation
- final print layout art
- locked character sheets

## Master Style Block

Warm premium early-reader children’s storybook illustration, charming rounded animated-style characters, expressive faces, soft natural lighting, colorful cozy Seed Valley plant-science world, magical but scientifically grounded, gentle textures, clear emotional storytelling, kid-friendly composition, no harsh realism, no scary details.

## Required Prompt Fields

Each prompt must include:

- Book ID
- Page number
- Exact page text reference
- Scene purpose
- Characters shown
- Character growth stage
- Character expressions
- Character placement
- Background/location
- Science detail
- Emotional beat
- Composition
- Lighting/mood
- Text-safe area
- Continuity notes
- Do-not-show notes

## Science Negative Rules

Do not show:
- Benny as a mature plant in Book 1.
- Shoot before root.
- Sunlight magically forcing germination underground.
- Flooding.
- Soil as food.
- Root drinking like a straw from a puddle.
- Every seed germinating the exact same way.

## Layout Negative Rules

Do not show:
- text over busy art
- important science detail hidden under text
- overcrowded full-cast scenes on every page
- random characters not in the manuscript
- harsh diagram style for story pages

## Approval Gate

A prompt is approved only when it matches:

1. Manuscript page purpose.
2. Science rules.
3. Character visual sheets.
4. Art Bible.
5. Page-by-page visual production map.
6. Text-safe layout requirements.
