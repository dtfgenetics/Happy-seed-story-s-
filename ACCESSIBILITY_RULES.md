# Accessibility Rules

Project ID: HSS-000  
Version: v0.1  
Status: Drafting  
Source of Truth: GitHub  

## Purpose

This file defines readability rules for books, PDFs, worksheets, and future digital products.

## Reading Rules

- Use clear readable fonts in final layout.
- Do not place text over busy art.
- Keep strong contrast between text and background.
- Keep story text blocks short enough for read-aloud pacing.
- Avoid cramped pages.
- Use consistent heading styles in back matter.
- Make worksheets printable in black and white when possible.

## Digital Rules

Future digital versions should include alt text, clear headings, logical page order, descriptive file names, and clear link text.

## Layout Rule

If text is hard to read, the art or layout must change.
