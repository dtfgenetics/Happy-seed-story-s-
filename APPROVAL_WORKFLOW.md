Project ID: HSS-000  
Version: v0.1

## Purpose

This file defines how project files move from idea to locked master.

## Status Flow

Idea -> Outlined -> Drafting -> In Review -> Needs Fixes -> Approved -> Locked -> Exported -> Published -> Archived

## Approval Gates

### Story Gate

The manuscript has a clear character problem, emotional arc, page turns, and satisfying ending.

### Science Gate

Science claims match SCIENCE_RULES.md and any required research is logged.

### Art Gate

Illustration plans match ART_BIBLE.md and the locked manuscript.

### Classroom Gate

Activities match CLASSROOM_SYSTEM.md, EXPERIMENT_TEMPLATE.md, and SAFETY_RULES.md.

### Publishing Gate

Publishing files match PUBLISHING_SYSTEM.md and metadata fields are complete enough for the current stage.

### Publicity Gate

Publicity files match PUBLICITY_SYSTEM.md and clearly speak to parents, teachers, libraries, and homeschool users.

## Locked Rule

A file can be marked Locked only when its GitHub file, Drive/Docs reference, and tracker status agree.
