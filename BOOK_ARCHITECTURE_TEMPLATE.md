# Book Architecture Template

Project ID: HSS-000  
Version: v0.1  
Status: Drafting  
Source of Truth: GitHub  

## Purpose

This file defines the repeatable 32-page structure for each Happy Seed Stories book.

## Standard 32-Page Direction

- Front cover concept
- Title page
- Dedication or opening page
- Main story pages
- Science back matter
- Vocabulary page
- Try-it-yourself activity
- Observation journal page
- Parent/teacher guide
- Next book teaser
- Back cover copy

## Page-Level Required Fields

Every page plan should include:

- Page number
- Story text
- Dialogue if needed
- Illustration direction
- Science purpose
- Emotional purpose
- Page-turn purpose
- Text-safe area notes

## Rule

The manuscript must be built before the final illustration prompts. Page art should never drift away from the locked text.
