# Character Continuity Rules

Project ID: HSS-000  
Version: v0.1  
Status: Drafting  
Source of Truth: GitHub  

## Purpose

This file protects character consistency across manuscripts, illustrations, classroom assets, and future books.

## Global Rules

- Each character keeps the same personality core across books.
- Visual design should not change unless the story clearly shows growth or transformation.
- A supporting character should not take over the main character's emotional arc.
- Each character has a science role and an emotional role.

## Book 1 Continuity

Benny Bean:

- begins as a seed
- swells after moisture
- cracks before root appears
- grows first root downward
- grows shoot upward
- ends as a tiny sprout
- does not become a mature plant in Book 1

Penny Pea:

- supports Benny through observation and questions
- should not become the Book 1 hero
- leads naturally into Book 2

Drip, Wally, and Sol:

- each teaches through action, not lectures
- should appear only where they support the page purpose

## Rule

If a character looks, acts, or teaches differently than the bible allows, fix the scene or update the bible before continuing.
