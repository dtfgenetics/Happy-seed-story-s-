Project ID: HSS-000  
Version: v0.1  
Status: Drafting  
Source of Truth: GitHub  

## Purpose

This file defines the repeatable classroom and home-learning system for every Happy Seed Stories book.

## Required Back Matter Per Book

Each book should include:

1. What We Learned
2. Words That Grew
3. Try It Yourself Activity
4. Observation Journal
5. Parent / Teacher Guide
6. Safety Notes
7. Next Adventure Teaser

## Activity Template

Every activity must include:

- Book ID
- Experiment title
- Science question
- Materials
- Steps
- Observation targets
- Expected range of results
- Safety notes
- Teacher notes
- Parent notes
- Extension idea
- Journal prompt

## Activity Rules

Activities should be:

- low-cost
- safe with adult supervision
- repeatable
- observable
- directly connected to the story
- possible at home or in a classroom
- focused on noticing, drawing, comparing, measuring, or describing

## Baseline Safety Language

Adult supervision recommended.  
Do not eat experiment seeds.  
Wash hands after handling seeds, soil, compost, or plant materials.  
Do not touch unknown outdoor plants.  
Use adult help for cutting fruit, stems, or plant samples.  
Avoid mold exposure. Do not touch mold directly.  
Check allergies before using food or plant materials.
