# Editorial Voice Guide

Project ID: HSS-000  
Version: v0.1  
Status: Drafting  
Source of Truth: GitHub  

## Purpose

This file defines the writing voice for Happy Seed Stories / Seed Valley so the series feels consistent across books.

## Voice Goals

- warm
- curious
- funny
- heartfelt
- clear
- musical without forced rhyming
- educational without sounding like a worksheet

## Audience Style

Primary reading experience: adult read-aloud with early-reader-friendly language.

Story pages should usually be simple and rhythmic. Back matter can be a little more explanatory.

## Dialogue Rules

Each character should sound distinct.

Benny: impatient, dramatic, funny, eager.  
Penny: curious, observant, calm, thoughtful.  
Drip: playful, bouncy, encouraging.  
Wally: practical, earthy, humorous.  
Sol: warm, patient, wise.  

## Science Language Rule

Science vocabulary may appear in the story when it feels natural, but it should be explained through context first and defined clearly in back matter.

## Avoid

- forced chants
- preachy morals
- long lectures
- overly babyish language
- jokes that distract from the story
- too many new science terms on one page

## Read-Aloud Rule

Every page should sound good when spoken aloud. If a sentence is awkward to read aloud, rewrite it.
