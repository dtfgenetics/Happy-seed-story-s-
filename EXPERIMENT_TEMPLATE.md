# Experiment Template

Project ID: HSS-000  
Version: v0.1  
Status: Drafting  
Source of Truth: GitHub  

## Purpose

This template standardizes every classroom and home activity in the series.

## Required Activity Structure

### Experiment Title

Clear, kid-friendly title.

### Book ID

Example: HSS-B01.

### Science Topic

One topic only.

### Science Question

Write as a question children can investigate.

### Materials

Low-cost, common, safe materials.

### Steps

Numbered and simple.

### Observation Targets

What should children look for, draw, count, measure, compare, or describe?

### Expected Range of Results

Do not guarantee exact outcomes. Explain what may happen.

### Safety Notes

Use SAFETY_RULES.md.

### Teacher Notes

How to use in a classroom.

### Parent Notes

How to use at home.

### Extension Activity

Optional deeper activity for older children.

### Observation Journal Prompt

A drawing or writing prompt tied to the activity.

## Rule

Activities are not meant to prove perfect results. They are meant to help children observe real changes carefully.
