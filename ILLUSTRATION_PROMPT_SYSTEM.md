# Illustration Prompt System

Project ID: HSS-000  
Version: v0.1  
Status: Drafting  
Source of Truth: GitHub  

## Purpose

This file defines how page-by-page image prompts should be written so illustrations stay connected to the approved manuscript.

## Prompt Rule

Every illustration prompt must be based on the locked page text and page purpose. Do not create random scenes that drift away from the story.

## Required Prompt Fields

- Book ID
- Page number
- Page text reference
- Scene purpose
- Characters shown
- Character placement
- Character expressions
- Background/location
- Science detail
- Emotional beat
- Composition notes
- Lighting/mood
- Text-safe area
- Continuity notes
- What should not appear
- Exact text that appears on the page, if any

## Style Anchor

Warm premium early-reader children's storybook style, rounded friendly animated characters, expressive faces, soft natural lighting, colorful Seed Valley environment, gentle magical plant-science feeling.

## Negative Continuity Rule

If a prompt conflicts with the science rules, character bible, art bible, or manuscript, it must be rewritten before image generation.
