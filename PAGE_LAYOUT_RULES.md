Project ID: HSS-000  
Version: v0.1

## Purpose

This file defines page layout rules before final design work begins.

## Current Layout Status

Trim size is not locked yet. Final art and final cover files should wait until trim size, page count, bleed, margins, and platform plan are selected.

## Storybook Structure Direction

Working structure:

- Front cover
- Title page
- Dedication or opening page
- Main story pages
- Science back matter
- Vocabulary page
- Try-it-yourself activity
- Observation journal
- Parent/teacher guide
- Next book teaser
- Back cover copy

## Text Placement Rules

- Keep text away from trim edges.
- Do not place body text over visually busy art.
- Every illustration prompt must include text-safe area notes.
- Page turns should guide composition.
- Important science visuals should not be hidden under text.

## Art/Layout Rule

The manuscript and page purpose control the image. Layout should support story clarity, not just decoration.

## Publishing Gate

Before final layout, update PUBLISHING_SYSTEM.md with the selected trim size, bleed, margins, page count, and output formats.
