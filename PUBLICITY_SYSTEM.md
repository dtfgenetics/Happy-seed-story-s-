## Purpose

This file defines how Happy Seed Stories / Seed Valley is positioned, promoted, and expanded beyond the manuscript.

## Core Pitch

Happy Seed Stories turns real plant science into warm, magical adventures where children learn through friendship, humor, emotional growth, and hands-on observation.

## Required Publicity Assets Per Book

Each book needs:

- one-sentence pitch
- parent pitch
- teacher pitch
- librarian pitch
- homeschool pitch
- back cover copy
- website product page copy
- short retail description
- long retail description
- launch social posts
- classroom freebie idea
- review request copy
- press release draft

## Audience Messages

Parent message: helps children understand nature, patience, curiosity, and growth.

Teacher message: classroom-ready read-aloud with vocabulary, observation, activity, and discussion support.

Library message: warm early-reader science story with strong read-aloud value and educational back matter.

Homeschool message: story-based plant science lesson with hands-on activity and journal prompt.

## Product Ladder

Free activity page -> single storybook -> storybook plus activity PDF -> Season 1 bundle -> teacher guide -> classroom experiment pack -> observation journal -> poster set -> vocabulary cards -> garden starter kit.

## Publicity Rule

Do not market the book only as cute. Market it as story-first plant science with parent and classroom usefulness.
