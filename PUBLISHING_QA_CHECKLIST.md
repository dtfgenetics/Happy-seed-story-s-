# Publishing QA Checklist

Project ID: HSS-000  
Version: v0.1  
Status: Drafting  
Source of Truth: GitHub  

## Purpose

This checklist reviews publishing readiness before layout, export, proofing, or launch.

## Format Checks

- Has the trim size been selected?
- Has the page count been confirmed?
- Have bleed, margins, and gutter been defined?
- Has paperback, hardcover, ebook, or PDF format been selected?
- Are final art dimensions based on the selected format?

## Metadata Checks

- Title is confirmed.
- Subtitle is confirmed or intentionally omitted.
- Series name and book number are listed.
- Author and illustrator credits are listed.
- Publisher or imprint field is prepared.
- Age range and grade range are included.
- Short and long descriptions are drafted.
- Keywords and categories are drafted.
- Back cover copy is drafted.

## Export Checks

- Interior export settings are known.
- Cover export settings are known.
- File naming follows FILE_NAMING_RULES.md.
- Proofing copy is stored in Drive.
- Tracker status matches the exported version.

## Rule

Do not create final print files until publishing specs are selected and researched.
