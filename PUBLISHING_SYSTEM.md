## Purpose

This file defines the publishing-control system for Happy Seed Stories / Seed Valley.

## Publishing Rule

Do not finalize illustrations, cover files, or layout exports until the publishing format is chosen.

## Required Publishing Decisions

Before final layout, define:

- trim size
- page count
- paperback / hardcover / ebook plan
- color interior standard
- bleed
- margins
- gutter
- cover dimensions
- barcode placement
- ISBN strategy
- imprint / publisher name
- copyright page language
- metadata
- price strategy
- platform plan

## Current Format Direction

Working assumption only, not locked:

- 32-page full-color children's picture book
- premium early-reader storybook feel
- print-ready interior later
- classroom PDF companions later

## Publishing Files Per Book

Each book needs:

- metadata file
- copyright page file
- back cover copy
- sales description
- keyword/category notes
- ISBN notes
- trim/bleed/margin notes
- export checklist
- proofing checklist
