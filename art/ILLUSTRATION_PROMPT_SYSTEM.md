# Illustration Prompt System

Project ID: HSS-000  
Version: v0.1  
Status: Drafting  
Source of Truth: GitHub  

## Purpose

This file defines the structure for page-by-page illustration prompts.

## Rule

Do not create final page prompts until the manuscript is approved. The manuscript controls the image plan.

## Master Prompt Components

Every prompt should include:

- Series style direction
- Book ID
- Page number
- Scene purpose
- Characters shown
- Character expressions
- Character placement
- Background / location
- Science detail
- Emotional beat
- Composition notes
- Lighting notes
- Text-safe area
- Continuity notes
- What should NOT appear
- Exact page text, if any

## Style Foundation

Premium early-reader children’s storybook, warm colorful animated-style characters, soft natural lighting, rounded friendly designs, expressive faces, magical plant-science world, clear educational structure.

## Negative Prompt / Do-Not-Show Rule

Page prompts must include what should not appear when there is a risk of scientific or story drift.

Example for Book 1:

Do not show Benny as a mature plant. Do not show sunlight magically forcing germination underground. Do not show the shoot before the first root.
