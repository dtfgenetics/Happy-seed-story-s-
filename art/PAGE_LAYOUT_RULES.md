# Page Layout Rules

Project ID: HSS-000  
Version: v0.1  
Status: Drafting  
Source of Truth: GitHub  

## Purpose

This file defines how text, art, page turns, and back matter should be structured.

## Layout Rule

Final layout cannot be locked until trim size, bleed, margins, and page count are chosen.

## Working Book Structure

- Front cover
- Title page
- Dedication / opening page
- Main story pages
- Science back matter
- Vocabulary
- Try-it-yourself activity
- Observation journal
- Parent / teacher guide
- Next book teaser
- Back cover

## Page Map Fields

Every page map should include:

- page number
- page type
- story text
- dialogue
- illustration direction
- science purpose
- emotional purpose
- text placement notes
- page-turn purpose
- continuity notes

## Text-Safe Rule

Important text and character faces should not be placed near trim edges, gutter, or busy background areas.

## Page-Turn Rule

Every spread should give the reader a reason to keep going: curiosity, surprise, humor, emotion, discovery, or anticipation.
