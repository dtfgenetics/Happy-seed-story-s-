## Purpose

This file defines how science vocabulary appears in the story and back matter.

## Vocabulary Rule

Science words should appear naturally in the story first, then be defined clearly in the back matter.

## Per-Book Target

Each book should include 6-10 useful vocabulary words.

## Rules

- Do not overload one page with too many new terms.
- Do not define words in a way that creates false science.
- Use the simplest accurate explanation.
- Back matter can be slightly more detailed than story pages.
