## Description Rules

Short description should be clear and buyer-facing.
Long description should explain story, science lesson, moral lesson, and classroom usefulness.
Back cover copy should be warm, polished, and readable by parents.

## Research Gate

Categories, ISBNs, trim, margins, bleed, and platform requirements must be researched before locking.
